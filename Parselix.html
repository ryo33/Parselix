Specs:

  • parser(term, String.t, source_parser) :: product_parser

Macro for creating a parser.

Result format which you can return in the function.

Example

parser "any" do
  fn
    _, _, "", position -> {:error, "EOF appeared.", position}
    _, _, x, _ -> {:ok, String.first(x), 1}
  end
end
